What is whitespace, HTML?

When we add alot of spaces beteen text in HTML, the browser will ignore it, and reduce it to single character spacing.

To get around this limitation we must add HTML entities    which add additional spacing, for a single space we use &nbsp;, and for a double space we use &ensp; and for a double, double space (4 single spaces) we use &emsp;.
